تنفيذ المخزن الدائري (Circular Buffer)
في هذا المشروع تم تنفيذ مخزن دائري (Circular Buffer) باستخدام لغة C 

 الوظائف
- تهيئة المخزن (Initialization)
- إدخال البيانات إلى المخزن (Write)
- قراءة/إزالة البيانات من المخزن (Read)
- التعامل مع حالات:
  - امتلاء المخزن (Overflow)
  - فراغ المخزن (Underflow)

يعتمد المخزن الدائري على:
- مؤشر head: لتحديد موقع القراءة
- مؤشر tail: لتحديد موقع الكتابة
- متغير count: لمعرفة عدد العناصر داخل المخزن

يتم تحريك المؤشرات باستخدام العملية:
(index + 1) % SIZE

وذلك للسماح بالعودة إلى بداية المصفوفة عند الوصول للنهاية.

آلية عمل البرنامج
1. إدخال اسم المستخدم من الإدخال القياسي
2. إضافة السلسلة "CE-ESY" إلى الاسم
3. تخزين الأحرف داخل المخزن الدائري
4. قراءة البيانات من المخزن وطباعة الناتج
5. التأكد من أن المخزن أصبح فارغ بعد القراءة
